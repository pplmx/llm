## Description
`mkdocs.yml` exists and configures Material, but there is no API
reference — readers cannot click a class name in a tutorial and land on
its docs. `mkdocstrings` renders docstrings as navigable pages with type
hints and cross-links. With our `src/` layout and `pydantic`-style
docstrings this is a high-leverage addition for onboarding.

## Acceptance criteria
- [ ] Add `mkdocstrings[python]>=0.26` to dev dependencies in

- [ ] Add `mkdocstrings-python` plugin to `mkdocs.yml` with handler
      options `docstring_style: google`, `show_source: true`,
      `show_root_heading: true`.
- [ ] New nav entry `📖 API Reference: reference/api/` generated by the
      plugin from explicit `::: llm.X` directives in
